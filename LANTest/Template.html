LAN Test Utility

Ask your friends to visit {{Address}} in their browsers.

If they cannot see this page, consider one of the following:

  • The interconnection between arbitary machines might not be permitted within your current network;
  • Your computer might not be discoverable on the network (e.g. you are running Windows in a public instead of private network):
    • Check Control Panel\All Control Panel Items\Network and Sharing Center\Advanced sharing settings for Network Sharing options.

Otherwise, there are configuration issues with your network router and it's beyond what this program can do. Try things like "Port Forwarding" among other solutions. (Most likely the router just doesn't allow LAN connection, consider using a different router)
In particular, if none of the games you have can be played in LAN mode, then you need to figure out how to replace/setup your router - and it might be related to ISP specific routers. Good luck!


P.S. In other cases, consider using a VPN network application like LogMeIn Hamachi to establish LAN connections between different devices. Or if you have a better open source alternative, please don't hesitate to email me and I will update the recommendation on this page.
